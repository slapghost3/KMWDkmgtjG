# 在线考试系统简介 python14

## 项目概述

在线考试系统是一个基于前后端分离架构的开发项目，使用 Django 和 MySQL 实现 Python 后端，以及 Vue.js 和 Element-UI 进行前端开发。此项目适合两位开发者合作完成，同时支持定制化开发需求。

## 技术栈

### 后端

- Python
- Django
- MySQL

### 前端

- Vue.js
- Element-UI

## 功能模块

功能模块涵盖在线考试的全流程，包括试题管理、考试管理、学生答题、成绩统计等。

## 系统角色

系统主要分为管理员、教师、学生三种角色。

## 运行环境

- 推荐使用 Python 3.*
- Django >= 2.2（如需可降级至 2.2 版本）
- Vue.js 2.*

## 部署步骤

1. 确保环境符合运行要求。
2. 根据提供的部署文档进行逐步部署。
3. 启动后端服务及前端服务。
